# Security Policy

## Supported Versions

Security updates will always only be applied to the latest version of Watch-Tower-NG.
As the software by default is set to auto-update if you use the `latest` tag, you will get these security updates automatically as soon as they are released.

## Reporting a Vulnerability

Critical vulnerabilities that might open up for external attacks should be reported via [GitHub Security Advisories](https://github.com/wiki-mod/Watch-Tower-NG/security/advisories/new).

Non-critical vulnerabilities may be reported as regular [GitHub issues](https://github.com/wiki-mod/Watch-Tower-NG/issues).

We will always try to get back to you as swiftly as possible. Keep in mind that since this is a community project, response time may vary.
